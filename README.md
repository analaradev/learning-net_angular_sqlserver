# learning-net_angular_sqlserver

Proyecto de aprendizaje full stack con Angular, .NET / .NET Core, C# y SQL Server.

## Categoría

`learning-` — práctica técnica para aprender un stack de desarrollo.

## Descripción

Repositorio orientado a practicar una aplicación web completa con frontend, backend y base de datos relacional.

## Tecnologías

- Angular
- TypeScript
- .NET / .NET Core
- C#
- SQL Server
- Git / GitHub

## Objetivos

- Practicar estructura full stack.
- Crear endpoints y controladores.
- Separar responsabilidades entre frontend, backend, servicios y repositorios.
- Conectar la aplicación con SQL Server.
- Documentar avances de aprendizaje.

## Estado

Proyecto de aprendizaje activo.
